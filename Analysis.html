Once that IÂ´ve built and tested my model itÂ´s time to use the 20 cases test set to predict the behavior of the people using the devices, first IÂ´m going to prepare the function to write the files that IÂ´m going to submit to Coursera.

pml_write_files = function(x, directory="solutionfiles"){
  dir.create (directory)
  n = length(x)
  for(i in 1:n){
    filename = paste0("problem_id_",i,".txt")
    filename=file.path(directory, filename)
    write.table(x[i],file=filename,quote=FALSE,row.names=FALSE,col.names=FALSE)
  }
}

Now that the function is written all I have to do is to use the predict function in the Random Forest model built using de TESTING data set provided (20 cases) and create the files, in the âsolutionfilesâ" directory provided in this repo are the 20 files that contains the prediction of each case provided in the TESTING data set.
